## 首页动画改造（framer-motion）

- 安装了 `framer-motion`（v12.36.0）作为动画库
- Hero 区域从 420/480px 拉到 `h-screen`（全屏高度，min 600px，max 1000px）
- Hero 背景图视差改用 framer-motion `useScroll` + `useTransform`：滚动时图片上移 + 缩放 + 遮罩渐隐
- Hero 文案（Badge / 标题 / 描述 / 搜索框）使用 stagger 序列入场动画（opacity + y + blur）
- Header 加入 fade-down 入场动画
- 底部新增 "Scroll" 滚动提示箭头（延迟出现 + 上下浮动循环动画）
- 工具卡片（ToolCard）使用 `motion.button` + `whileInView` 滚入淡入，stagger per row
- 分类标题（SectionTitle）使用 `motion.div` + `whileInView` 左滑入
- 空搜索结果加入 scale + opacity 入场动画
- 标题字号放大：sm:5xl → lg:6xl
