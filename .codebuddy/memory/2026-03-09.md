## React 最佳实践优化 — 全量修复

基于 Vercel React Best Practices 规范，对 qiao-tools-x 项目进行了全面优化。修改了以下文件：

### P0: Bundle Size
- `vite.config.ts` — 添加 `babel-plugin-direct-import` 优化 lucide-react barrel import、移除 `vendor-icons` chunk、添加 `build.target: 'es2022'` 和 `cssCodeSplit`

### P0: Zustand Selector
- `src/pages/qrcode-tool/index.tsx` — 全量订阅改为逐个 selector 订阅
- `src/pages/markdown-editor/index.tsx` — 全量订阅改为逐个 selector 订阅

### P1: Markdown Editor
- ZOOM 常量从组件函数体提升到模块级别
- 滚动进度使用 `startTransition` 包裹
- useCallback 依赖数组清理

### P1: SSE-to-JSON
- SseBlock 的 `onToggle` 改为接收 `index` 参数，内部用 `useCallback` 包装，让 `memo` 生效
- `collapseAll` 使用 `blocksRef` 避免 `blocks` 依赖

### P2: useIsMobile + App + Menu Store
- `useIsMobile` 使用懒初始化消除首帧闪烁
- `menu store` 移除 `isPinned` 方法，`togglePin` 改为纯 `set` 函数式更新
- `App.tsx` 分组逻辑合并为单次遍历 + `Set` 查找

### P2: Image Compressor
- 17 个 `useState` 重构为 `useReducer`

- IIFE `compressRatioText` 改为 `useMemo`

- `toggleAspectRatio` 修复 stale closure

### P3: Store 清理
- `qrcode-tool/store.ts` 和 `markdown-editor/store.ts` 移除多余 `createJSONStorage`
